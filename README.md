## Development

To set up local development:

### 1. Install dependencies

```sh
npm i
```

### 2. Start development server

To start the development server at <http://localhost:3000> run:

```sh
npm run dev
```

### 3. Deploy to GitHub Pages
To deploy your changes to gh-pages run:

```sh
npm run deploy
```
